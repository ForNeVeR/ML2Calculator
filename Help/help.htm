Вступление

Программа Вычислитель предназначена не только для элементарных вычислений. В неё также заложены функции синус, косинус, тангенс и котангенс, она может возводить в произвольную степень и извлекать корни произвольной степени, выдаёт n-ый член арифметической прогрессии и сумму первых n её членов.

Описание интерфейса

Окно программы
Интерфейс программы состоит из нескольких закладок, переключаться между которыми можно в верхней части экрана.

Эти закладки: Решение квадратных уравнений, Решение обычных уравнений, Калькулятор, Справка, Прогрессия. Далее каждая из них будет рассмотрена более подробно.

Закладка "Решение квадратных уравнений"

Закладка "Решение квадратных уравнений"
На этой закладке Вы можете решить квадратные уравнения, т.е. уравнения вида ax^2+bx+c=0. Введите коэффициенты a, b и c и нажмите кнопку "Решить!", после чего будет выведено сообщение со значениями корней. Если какие-либо данные неаерны (т.е. уравнение не может быть решено) или ввод неверен (в полях находятся посторонние знаки, буквы и т.д.), то будет выведено соответствующее сообщение.

Закладка "Решение обычных уравнений"

Закладка "Решение обычных уравнений"
С помощью этой закладки Вы можете решить уравнения вида ax+b=0. Введите значения коэффициентов и нажмите кнопку "Решить!", и Вы получите окно с результатом или сообщением об ошибке.

Закладка "Калькулятор"

Закладка "Калькулятор"
Самая содержательная закладка. В следующих версиях, похоже, придётся увеличивать окно программы для предоставления этой закладке большей площади. Итак. Закладка состоит из нескольких блоков. Справа - блок, управляющий десятичным разделителем (знаком, отделяющим целую часть числа от дробной). Можно ставить точку или запятую. Можно сохранить настройку, чтобы при следующем запуске программы она была считана автоматически. Есть блоки с цифрами и стандартными действиями. Тут всё просто - как в обычном калькуляторе. Можно возводить числа в любую (даже дробную) степень и извлекать любые корни. Есть блок тригонометрических функций (упомянутые выше синус, косинус и др.). Однако пока значеня углов следует вводить только в радианах. В следующих версиях это досадное недоразумение будет исправлено. И, наконец, есть блок работы с памятью. Там можно прибавлять число из поля ввода в память, отнимать оттуда и наоборот, вводить число из памяти в поле ввода.

Закладка "Справка"

Закладка "Справка"
Тут Вы увидите краткую справку по некоторым закладкам, а также кнопку вызова данного файла.

Закладка "Прогрессия"

Закладка "Прогрессия"
На этой закладке Вы, после ввода нужных данных, получите n-ый элемент арифметической прогрессии и сумму первых n элементов прогрессии. Немного, но пригодиться может.
